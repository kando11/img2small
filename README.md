"# 🖼️ Image Compressor

A simple, minimalist, and powerful web-based image compression tool that runs entirely in your browser!

## ✨ Features

- **🎨 Beautiful UI**: Clean, modern, and user-friendly interface
- **⚡ Bulk Processing**: Compress multiple images at once
- **🔒 Privacy First**: All processing happens client-side (no server upload)
- **📊 Real-time Progress**: Minimalist progress bar with detailed status
- **💾 Smart Compression**: Adjustable quality, max width, and output format
- **📱 Responsive**: Works perfectly on desktop and mobile
- **🚀 Zero Dependencies**: Pure HTML, CSS, and JavaScript

## 🎯 How to Use

### Option 1: GitHub Pages (Recommended)
1. Go to your repository settings
2. Navigate to "Pages" section
3. Select your branch and save
4. Access your app at `https://yourusername.github.io/repo-name`

### Option 2: Open Locally
Simply open `index.html` in any modern web browser

### Option 3: Drag & Drop
1. Drag and drop images onto the drop zone
2. Or click to browse and select multiple images
3. Adjust compression settings (quality, max width, format)
4. Click "Compress Images"
5. Download all compressed files automatically!

## ⚙️ Settings

- **Quality (1-100%)**: Control the compression level
- **Max Width**: Resize images to a maximum width (optional)
- **Output Format**: Choose between JPEG, PNG, WebP, or keep original

## 🛠️ Tech Stack

- Pure HTML5
- CSS3 with modern features
- Vanilla JavaScript (ES6+)
- Canvas API for image processing
